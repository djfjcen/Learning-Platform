=== 知识点ID: 7
=== 知识点名称: 集合表示法

## EXERCISE
题型: SINGLE_CHOICE
难度: EASY
类别: BASIC
题目: 下列用列举法表示集合正确的是？
选项: ["A. A = {1, 1, 2, 3}", "B. A = {1, 2, 3}", "C. A = {x | x 是正整数}", "D. A = {1, 2, ...}"]
答案: B
解析: 列举法要求将元素一一列在大括号内，元素不能重复（A重复了1），C是描述法，D省略号不完整。

## EXERCISE
题型: SINGLE_CHOICE
难度: EASY
类别: BASIC
题目: 集合 {3, 1, 2} 与 {1, 2, 3} 的关系是？
选项: ["A. 不相等", "B. 相等", "C. 前者是后者的真子集", "D. 无法比较"]
答案: B
解析: 集合中的元素是无序的，{3, 1, 2} 和 {1, 2, 3} 包含相同的元素，因此是同一个集合。

## EXERCISE
题型: MULTIPLE_CHOICE
难度: MEDIUM
类别: BASIC
题目: 下列关于集合表示法说法正确的有？
选项: ["A. 列举法适合元素个数有限且较少的集合", "B. 描述法的一般形式为 {x | P(x)}", "C. 文氏图可以严格表示集合", "D. 同一集合可以用列举法和描述法分别表示"]
答案: A,B,D
解析: 文氏图只是辅助可视化工具，不能严格表示集合（C错误）。A、B、D均为正确描述。

## EXERCISE
题型: FILL_BLANK
难度: EASY
类别: BASIC
题目: 用描述法表示所有大于5的整数组成的集合，应写为___。
答案: {x | x > 5 且 x 为整数}
解析: 描述法的一般形式为 {x | P(x)}，P(x) 是元素满足的条件。大于5的整数需同时满足"大于5"和"是整数"两个条件。

## EXERCISE
题型: SINGLE_CHOICE
难度: MEDIUM
类别: POSTGRADUATE
题目: 设集合 A = {x | x² - 3x + 2 = 0}，则 A 用列举法表示为？
选项: ["A. {1}", "B. {2}", "C. {1, 2}", "D. {-1, -2}"]
答案: C
解析: 解方程 x² - 3x + 2 = 0，得 (x-1)(x-2) = 0，所以 x = 1 或 x = 2，A = {1, 2}。

=== 知识点ID: 8
=== 知识点名称: 元素隶属关系

## EXERCISE
题型: SINGLE_CHOICE
难度: EASY
类别: BASIC
题目: 设 A = {1, 2, 3}，下列关系正确的是？
选项: ["A. 1 ⊂ A", "B. {1} ∈ A", "C. 1 ∈ A", "D. ∅ ∈ A"]
解析: 1是A的元素，用∈表示属于关系。⊂用于集合间的包含关系，{1}是集合不是元素，∅是空集不是A的元素。
答案: C

## EXERCISE
题型: SINGLE_CHOICE
难度: MEDIUM
类别: POSTGRADUATE
题目: 设 A = {1, 2, {3}}，下列说法正确的是？
选项: ["A. 3 ∈ A", "B. {3} ⊂ A", "C. {3} ∈ A", "D. {1, 2} ∈ A"]
答案: C
解析: A的元素是1、2和{3}，所以{3} ∈ A正确。3不是A的直接元素（A中没有3这个元素），{3} ⊂ A也错误，因为3 ∉ A。

## EXERCISE
题型: MULTIPLE_CHOICE
难度: MEDIUM
类别: BASIC
题目: 下列关于子集关系正确的有？
选项: ["A. ∅ ⊆ {1, 2}", "B. {1} ⊆ {1, 2}", "C. {1, 2} ⊆ {1, 2}", "D. {1, 2} ⊂ {1, 2}"]
答案: A,B,C
解析: 空集是任何集合的子集（A正确），{1}的每个元素都属于{1,2}（B正确），任何集合是自身的子集（C正确）。真子集要求不相等，{1,2}不是自身的真子集（D错误）。

## EXERCISE
题型: FILL_BLANK
难度: EASY
类别: BASIC
题目: 若 A ⊆ B 且 B ⊆ A，则 A ___ B。
答案: =
解析: 集合相等的定义：若A是B的子集且B是A的子集，则A与B相等。

## EXERCISE
题型: FILL_BLANK
难度: MEDIUM
类别: POSTGRADUATE
题目: 设全集 U = {1, 2, 3, 4, 5}，A = {1, 2, 3}，则 A 的补集 Ā = ___。
答案: {4,5}
解析: 补集 Ā = U - A = {4, 5}，即全集中不属于A的元素。

=== 知识点ID: 9
=== 知识点名称: 集合运算

## EXERCISE
题型: SINGLE_CHOICE
难度: EASY
类别: BASIC
题目: 设 A = {1, 2, 3}，B = {2, 3, 4}，则 A ∪ B = ?
选项: ["A. {2, 3}", "B. {1, 2, 3, 4}", "C. {1}", "D. {4}"]
答案: B
解析: 并集 A ∪ B 包含属于A或属于B的所有元素，即 {1, 2, 3, 4}。

## EXERCISE
题型: SINGLE_CHOICE
难度: EASY
类别: BASIC
题目: 设 A = {1, 2, 3}，B = {2, 3, 4}，则 A ∩ B = ?
选项: ["A. {1, 2, 3, 4}", "B. {2, 3}", "C. {1, 4}", "D. ∅"]
答案: B
解析: 交集 A ∩ B 包含同时属于A和B的元素，即 {2, 3}。

## EXERCISE
题型: SINGLE_CHOICE
难度: MEDIUM
类别: BASIC
题目: 设 A = {1, 2, 3}，B = {2, 3, 4}，则 A - B = ?
选项: ["A. {1}", "B. {4}", "C. {1, 4}", "D. {2, 3}"]
答案: A
解析: 差集 A - B 包含属于A但不属于B的元素，2和3都在B中，只有1不在B中，所以 A - B = {1}。

## EXERCISE
题型: MULTIPLE_CHOICE
难度: MEDIUM
类别: BASIC
题目: 下列集合运算律正确的有？
选项: ["A. A ∪ A = A（幂等律）", "B. A ∩ ∅ = ∅（零律）", "C. A ∪ B = B ∪ A（交换律）", "D. A - B = B - A（交换律）"]
答案: A,B,C
解析: 差集不满足交换律，A - B ≠ B - A（一般情况下），D错误。A、B、C均为正确的集合运算律。

## EXERCISE
题型: FILL_BLANK
难度: MEDIUM
类别: POSTGRADUATE
题目: 设 A = {1, 2}，B = {a, b}，则 A × B 的元素个数为___，A × B = ___。
答案: 4,{(1,a),(1,b),(2,a),(2,b)}
解析: 笛卡尔积 |A × B| = |A| × |B| = 2 × 2 = 4。A × B = {(1,a), (1,b), (2,a), (2,b)}。

## EXERCISE
题型: SINGLE_CHOICE
难度: HARD
类别: INTERVIEW
题目: 设 A = {1, 2, 3}，B = {3, 4, 5}，则 A △ B（对称差）等于？
选项: ["A. {3}", "B. {1, 2, 4, 5}", "C. {1, 2, 3, 4, 5}", "D. ∅"]
答案: B
解析: 对称差 A △ B = (A - B) ∪ (B - A) = {1, 2} ∪ {4, 5} = {1, 2, 4, 5}。

=== 知识点ID: 10
=== 知识点名称: 集合存储实现

## EXERCISE
题型: SINGLE_CHOICE
难度: EASY
类别: BASIC
题目: 在无序数组中查找某个元素的时间复杂度是？
选项: ["A. O(1)", "B. O(log n)", "C. O(n)", "D. O(n²)"]
答案: C
解析: 无序数组没有排序，只能从头到尾逐个比较，最坏情况需要遍历整个数组，时间复杂度 O(n)。

## EXERCISE
题型: SINGLE_CHOICE
难度: MEDIUM
类别: BASIC
题目: 在哈希表中查找元素的平均时间复杂度是？
选项: ["A. O(n)", "B. O(log n)", "C. O(1)", "D. O(n log n)"]
答案: C
解析: 哈希表通过哈希函数直接定位元素位置，平均情况下查找时间复杂度为 O(1)，但最坏情况（所有元素哈希冲突）退化为 O(n)。

## EXERCISE
题型: MULTIPLE_CHOICE
难度: MEDIUM
类别: INTERVIEW
题目: 下列关于集合存储实现说法正确的有？
选项: ["A. 有序数组支持二分查找，查找效率 O(log n)", "B. 链表支持随机访问", "C. 位图适合存储小整数集合", "D. 哈希表最坏情况下查找复杂度为 O(n)"]
答案: A,C,D
解析: 链表不支持随机访问（B错误），必须从头遍历。A、C、D均为正确描述。

## EXERCISE
题型: FILL_BLANK
难度: MEDIUM
类别: POSTGRADUATE
题目: 哈希表处理冲突的两种常见方法是___和___。
答案: 开放寻址法,链地址法
解析: 开放寻址法在冲突时寻找下一个空位（如线性探测），链地址法将冲突元素用链表串联在同一个桶中。

## EXERCISE
题型: SINGLE_CHOICE
难度: HARD
类别: INTERVIEW
题目: 当集合元素为 0~99 的整数时，最节省空间且高效的存储方式是？
选项: ["A. 有序数组", "B. 链表", "C. 哈希表", "D. 位图"]
答案: D
解析: 位图只需 100 个比特位（约13字节），每个整数对应一个比特位，插入/查找/删除均为 O(1)，是最节省空间且高效的方案。

## EXERCISE
题型: FILL_BLANK
难度: HARD
类别: INTERVIEW
题目: 用位图存储 0~999 范围内的整数集合，至少需要___字节的存储空间。
答案: 125
解析: 0~999共1000个整数，每个占1个比特位，共需 1000/8 = 125 字节。
