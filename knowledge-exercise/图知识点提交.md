=== 知识点 ID: 19
=== 知识点名称：图的表示

## OPERATION
标题：邻接矩阵表示法
内容：
用二维数组表示图中顶点之间的关系。
对于无向图，若顶点 i 和 j 之间有边，则 matrix[i][j] = matrix[j][i] = 1，否则为 0。
对于有向图，若从 i 到 j 有弧，则 matrix[i][j] = 1。
适合存储稠密图，空间复杂度 O(n²)，判断两顶点间是否有边的时间复杂度为 O(1)。

标题：邻接表表示法
内容：
对每个顶点建立一个单链表，存储与该顶点相邻接的所有顶点。
无向图中，顶点 v 的链表包含所有与 v 相连的顶点；有向图中，链表包含从 v 出发能到达的顶点。
适合存储稀疏图，空间复杂度 O(n+e)，其中 n 为顶点数，e 为边数。
查找某顶点的邻接点效率高，但判断两顶点间是否有边需遍历链表。

## COMPLEXITY
标题：存储结构复杂度对比
内容：
邻接矩阵：
- 空间复杂度：O(n²)，与边数无关
- 判断边是否存在：O(1)
- 获取顶点的度：O(n)
- 适合稠密图

邻接表：
- 空间复杂度：O(n+e)
- 判断边是否存在：O(degree)
- 获取顶点的度：O(1)（若维护度计数器）
- 适合稀疏图

## CODE
标题：邻接矩阵实现
描述：用二维数组存储无向图
语言：C++
代码：
#include <bits/stdc++.h>
using namespace std;

class Graph {
    int n;
    vector<vector<int>> matrix;
public:
    Graph(int n) : n(n), matrix(n, vector<int>(n, 0)) {}
    
    void addEdge(int i, int j) {
        if (i >= 0 && i < n && j >= 0 && j < n) {
            matrix[i][j] = matrix[j][i] = 1;
        }
    }
    
    bool hasEdge(int i, int j) {
        return i >= 0 && i < n && j >= 0 && j < n && matrix[i][j] == 1;
    }
    
    void print() {
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < n; j++) {
                cout << matrix[i][j] << " ";
            }
            cout << endl;
        }
    }
};

int main() {
    Graph g(4);
    g.addEdge(0, 1);
    g.addEdge(0, 2);
    g.addEdge(1, 3);
    g.addEdge(2, 3);
    cout << "邻接矩阵:" << endl;
    g.print();
    cout << "0 和 2 之间有边吗？" << (g.hasEdge(0, 2) ? "是" : "否") << endl;
    return 0;
}

=== 知识点 ID: 20
=== 知识点名称：图的遍历

## OPERATION
标题：深度优先搜索（DFS）
内容：
从起始顶点出发，沿一条路径一直访问下去，直到无法继续，然后回溯到上一个顶点，继续访问其他未访问的邻接点。
递归实现：访问当前顶点，标记为已访问，然后递归访问所有未访问的邻接点。
非递归实现：使用栈，将起始顶点入栈，每次取出栈顶顶点，若未访问则访问并入栈其邻接点。
遍历序列不唯一，取决于访问邻接点的顺序。

标题：广度优先搜索（BFS）
内容：
从起始顶点出发，先访问所有邻接点，然后再依次访问这些邻接点的邻接点，层次式扩展。
使用队列实现：起始顶点入队，每次取出队首顶点并访问，将其未访问的邻接点入队。
适合求解最短路径问题（无权图），遍历序列不唯一。

## COMPLEXITY
标题：遍历算法复杂度
内容：
DFS 和 BFS 的时间复杂度相同：
- 邻接矩阵存储：O(n²)，需要遍历矩阵的行
- 邻接表存储：O(n+e)，每个顶点和每条边各访问一次

空间复杂度：
- DFS：O(n)，递归栈深度或显式栈空间
- BFS：O(n)，队列空间

两者都能遍历图中所有可达顶点，时间复杂度相同，但应用场景不同。

## CODE
标题：DFS 和 BFS 遍历实现
描述：分别用递归 DFS 和队列 BFS 遍历图
语言：C++
代码：
#include <bits/stdc++.h>
using namespace std;

class Graph {
    int n;
    vector<vector<int>> adj;
    vector<bool> visited;
public:
    Graph(int n) : n(n), adj(n), visited(n, false) {}
    
    void addEdge(int i, int j) {
        adj[i].push_back(j);
        adj[j].push_back(i);
    }
    
    void dfs(int v) {
        visited[v] = true;
        cout << v << " ";
        for (int neighbor : adj[v]) {
            if (!visited[neighbor]) {
                dfs(neighbor);
            }
        }
    }
    
    void bfs(int start) {
        fill(visited.begin(), visited.end(), false);
        queue<int> q;
        q.push(start);
        visited[start] = true;
        
        while (!q.empty()) {
            int v = q.front();
            q.pop();
            cout << v << " ";
            
            for (int neighbor : adj[v]) {
                if (!visited[neighbor]) {
                    visited[neighbor] = true;
                    q.push(neighbor);
                }
            }
        }
    }
};

int main() {
    Graph g(5);
    g.addEdge(0, 1);
    g.addEdge(0, 2);
    g.addEdge(1, 3);
    g.addEdge(2, 3);
    g.addEdge(3, 4);
    
    cout << "DFS 从 0 开始：";
    g.dfs(0);
    cout << endl;
    
    cout << "BFS 从 0 开始：";
    g.bfs(0);
    cout << endl;
    
    return 0;
}

=== 知识点 ID: 21
=== 知识点名称：最短路径

## OPERATION
标题：Dijkstra 算法
内容：
用于求解单源最短路径，即从源点到图中所有其他顶点的最短路径。
贪心策略：每次选择距离源点最近且未访问过的顶点，更新其邻接点的距离。
要求图中边的权值非负。
使用优先队列优化后，时间复杂度为 O((n+e)log n)。

标题：Floyd 算法
内容：
用于求解所有顶点对之间的最短路径。
动态规划思想：逐步考虑每个中间顶点，更新任意两点间的最短距离。
三重循环，时间复杂度 O(n³)，适合小规模图或稠密图。
可以处理负权边，但不能处理负权回路。

## COMPLEXITY
标题：最短路径算法复杂度
内容：
Dijkstra 算法：
- 时间复杂度：O(n²)（朴素实现）或 O((n+e)log n)（优先队列优化）
- 空间复杂度：O(n)
- 适用：单源最短路径，边权非负

Floyd 算法：
- 时间复杂度：O(n³)
- 空间复杂度：O(n²)
- 适用：所有顶点对最短路径，可处理负权边

## CODE
标题：Dijkstra 算法实现
描述：用优先队列优化求单源最短路径
语言：C++
代码：
#include <bits/stdc++.h>
using namespace std;

const int INF = 1e9;

class Graph {
    int n;
    vector<vector<pair<int, int>>> adj;
public:
    Graph(int n) : n(n), adj(n) {}
    
    void addEdge(int u, int v, int w) {
        adj[u].push_back({v, w});
        adj[v].push_back({u, w});
    }
    
    vector<int> dijkstra(int start) {
        vector<int> dist(n, INF);
        dist[start] = 0;
        priority_queue<pair<int, int>, vector<pair<int, int>>, greater<pair<int, int>>> pq;
        pq.push({0, start});
        
        while (!pq.empty()) {
            int d = pq.top().first;
            int u = pq.top().second;
            pq.pop();
            
            if (d > dist[u]) continue;
            
            for (auto& edge : adj[u]) {
                int v = edge.first;
                int w = edge.second;
                if (dist[u] + w < dist[v]) {
                    dist[v] = dist[u] + w;
                    pq.push({dist[v], v});
                }
            }
        }
        return dist;
    }
};

int main() {
    Graph g(5);
    g.addEdge(0, 1, 4);
    g.addEdge(0, 2, 1);
    g.addEdge(1, 3, 1);
    g.addEdge(2, 1, 2);
    g.addEdge(2, 3, 5);
    g.addEdge(3, 4, 3);
    
    vector<int> dist = g.dijkstra(0);
    
    cout << "从顶点 0 到各点的最短距离：" << endl;
    for (int i = 0; i < 5; i++) {
        if (dist[i] == INF)
            cout << "到 " << i << " : 不可达" << endl;
        else
            cout << "到 " << i << " : " << dist[i] << endl;
    }
    
    return 0;
}

=== 知识点 ID: 22
=== 知识点名称：最小生成树

## OPERATION
标题：Prim 算法
内容：
贪心策略：从任意顶点开始，每次选择连接已选顶点集和未选顶点集的最小权值边，将该边连接的未选顶点加入已选集。
适合稠密图，使用邻接矩阵存储时时间复杂度为 O(n²)。
使用优先队列优化后，时间复杂度为 O((n+e)log n)。

标题：Kruskal 算法
内容：
贪心策略：将所有边按权值从小到大排序，依次选择最小边，若该边连接的两个顶点不在同一连通分量中，则加入生成树。
使用并查集判断顶点是否在同一连通分量。
适合稀疏图，时间复杂度 O(e log e)。

## COMPLEXITY
标题：最小生成树算法复杂度
内容：
Prim 算法：
- 时间复杂度：O(n²)（朴素实现）或 O((n+e)log n)（优先队列优化）
- 空间复杂度：O(n)
- 适合稠密图

Kruskal 算法：
- 时间复杂度：O(e log e)，主要消耗在边排序
- 空间复杂度：O(n)，并查集空间
- 适合稀疏图

两种算法都能得到相同的最小生成树权值和，但树的形态可能不同。

## CODE
标题：Kruskal 算法实现
描述：用并查集求最小生成树
语言：C++
代码：
#include <bits/stdc++.h>
using namespace std;

struct Edge {
    int u, v, w;
    bool operator<(const Edge& other) const {
        return w < other.w;
    }
};

class UnionFind {
    vector<int> parent;
public:
    UnionFind(int n) {
        parent.resize(n);
        for (int i = 0; i < n; i++)
            parent[i] = i;
    }
    
    int find(int x) {
        if (parent[x] != x)
            parent[x] = find(parent[x]);
        return parent[x];
    }
    
    bool unite(int x, int y) {
        int rootX = find(x);
        int rootY = find(y);
        if (rootX != rootY) {
            parent[rootX] = rootY;
            return true;
        }
        return false;
    }
};

class Graph {
    int n;
    vector<Edge> edges;
public:
    Graph(int n) : n(n) {}
    
    void addEdge(int u, int v, int w) {
        edges.push_back({u, v, w});
    }
    
    int kruskal() {
        sort(edges.begin(), edges.end());
        UnionFind uf(n);
        int mstWeight = 0;
        int edgesCount = 0;
        
        for (const auto& edge : edges) {
            if (uf.unite(edge.u, edge.v)) {
                mstWeight += edge.w;
                edgesCount++;
                if (edgesCount == n - 1)
                    break;
            }
        }
        
        return edgesCount == n - 1 ? mstWeight : -1;
    }
};

int main() {
    Graph g(5);
    g.addEdge(0, 1, 2);
    g.addEdge(0, 2, 3);
    g.addEdge(1, 2, 1);
    g.addEdge(1, 3, 4);
    g.addEdge(2, 3, 5);
    g.addEdge(3, 4, 6);
    
    int mst = g.kruskal();
    cout << "最小生成树的权值和：" << (mst != -1 ? to_string(mst) : "无法生成") << endl;
    
    return 0;
}
