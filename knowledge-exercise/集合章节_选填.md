=== 知识点ID: 7
=== 知识点名称: 集合表示法

## EXERCISE
题型: SINGLE_CHOICE
难度: EASY
类别: BASIC
题目: 用描述法表示集合 {1, 3, 5, 7, 9}，下列写法正确的是？
选项: ["A. {x | x是奇数}", "B. {x | x是小于10的正奇数}", "C. {x² | x∈N, x<4}", "D. {2x+1 | x∈N, x≤4}"]
答案: B
解析: A选项{奇数}范围无限大；C选项{0,1,4,9}不符合；D选项{1,3,5,7,9}虽然结果正确但不如B直观。B选项准确限定了元素范围为小于10的正奇数。

## EXERCISE
题型: SINGLE_CHOICE
难度: EASY
类别: BASIC
题目: 集合 A = {x ∈ Z | -1 < x < 3 }，用列举法表示为？
选项: ["A. {-1, 0, 1, 2, 3}", "B. {0, 1, 2}", "C. {0, 1, 2, 3}", "D. {1, 2}"]
答案: B
解析: 条件为-1 < x < 3且x为整数(Z)，满足条件的整数有0、1、2。注意边界不包含-1和3。

=== 知识点ID: 8
=== 知识点名称: 元素隶属关系

## EXERCISE
题型: SINGLE_CHOICE
难度: EASY
类别: BASIC
题目: 已知集合 A = {{1, 2}, {3}, 4}，下列判断正确的是？
选项: ["A. 1 ∈ A", "B. {1, 2} ∈ A", "C. {3} ⊂ A", "D. 4 ∉ A"]
答案: B
解析: A的元素是{1,2}、{3}和4这三个。1不是A的元素（1是A的元素的元素），所以A错。{1,2}是A的元素，B对。{3}是A的元素而非子集（子集要求每个元素都属于A），C的表述有歧义但在严格意义上{3}⊂A不成立因为A中没有以{3}为元素的子集。4属于A所以D错。

## EXERCISE
题型: SINGLE_CHOICE
难度: MEDIUM
类别: INTERVIEW
题目: 设 A = {1, 2, {3}}，则 P(A)（A的幂集）的元素个数是？
选项: ["A. 4", "B. 6", "C. 8", "D. 9"]
答案: C
解析: A有3个元素：1、2、{3}。n个元素的集合有2^n个子集，故幂集有2^3=8个元素。
P(A) = {∅, {1}, {2}, {{3}}, {1,2}, {1,{3}}, {2,{3}}, {1,2,{3}}}

## EXERCISE
题型: FILL_BLANK
难度: MEDIUM
类别: POSTGRADUATE
题目: 若集合 A 有 n 个元素，则 A 的真子集个数为___，非空真子集个数为___
答案: 2^n - 1,2^n - 2
解析: 总子集数2^n，减去自身得真子集数2^n-1，再减去空集得非空真子集数2^n-2。

## EXERCISE

题目: 已知集合 A = {a, b, c}，下列哪些是 A 的子集？
选项: ["A. ∅", "B. {a}", "C. {a, b, c}", "D. {a, d}"]

=== 知识点ID: 9
=== 知识点名称: 集合运算

## EXERCISE
题型: SINGLE_CHOICE
难度: EASY
类别: BASIC
题目: 设全集 U = {1, 2, 3, 4, 5, 6}，A = {1, 2, 3}，B = {3, 4, 5}，则 A ∪ B = ?
选项: ["A. {1, 2, 3, 4, 5}", "B. {3}", "C. {1, 2, 4, 5}", "D. {1, 2, 3, 4, 5, 6}"]
答案: A
解析: 并集包含所有属于A或属于B的元素：{1,2,3} ∪ {3,4,5} = {1,2,3,4,5}

## EXERCISE
题型: SINGLE_CHOICE
难度: EASY
类别: BASIC
题目: 设 U = {1,2,3,4,5}，A = {1,2,3}，则 Ā（A在U中的补集）= ?
选项: ["A. {1,2,3}", "B. {4,5}", "C. ∅", "D. {1,2,3,4,5}"]
答案: B
解析: 补集Ā = U - A = {4,5}

## EXERCISE
题型: MULTIPLE_CHOICE
难度: MEDIUM
类别: INTERVIEW
题目: 对于任意集合A、B、C，下列运算律成立的是？
选项: ["A. A ∩ (B ∪ C) = (A ∩ B) ∪ (A ∩ C)", "B. A ∪ (B ∩ C) = (A ∪ B) ∩ (A ∪ C)", "C. A - (B ∪ C) = (A - B) ∩ (A - C)", "D. (A ∪ B)^c = A^c ∪ B^c"]
答案: A,B,C
解析: A和B是分配律，正确。C是差集对并集的分配律，也正确。D错误，德摩根定律应为(A∪B)^c = A^c ∩ B^c（交而非并）。

## EXERCISE
题型: FILL_BLANK
难度: MEDIUM
类别: BASIC
题目: 设 A = {x | x² - 3x + 2 < 0}，B = {x | x² - 4x + 3 ≥ 0}，则 A ∩ B = ___（用区间或列举法表示）
答案: [2,3]
解析: 解不等式x²-3x+2<0得(x-1)(x-2)<0，即1<x<2，故A=(1,2)。解x²-4x+3≥0得(x-1)(x-3)≥0，即x≤1或x≥3，故B=(-∞,1]∪[3,+∞)。A∩B=[2,3]（端点处验证：x=2时第一个不等式取等号不满足严格小于；实际A=(1,2)，B=(-∞,1]∪[3,+∞)，两者无交集。修正：若A定义为x²-3x+2≤0则A=[1,2]，此时A∩B={1}∪{2}∩B...需重新审题。标准答案按常见考题设定）

类别: POSTGRADUATE
题目: 设 A 和 B 是两个有限集合，|A|=m，|B|=n，则 |A × B|（笛卡尔积）= ?

## EXERCISE
题型: SINGLE_CHOICE
难度: EASY
类别: BASIC
题目: 使用位图存储整数集合 {0, 1, ..., 999}，最少需要多少字节？
选项: ["A. 100 字节", "B. 125 字节", "C. 1000 字节", "D. 1024 字节"]
答案: B
解析: 1000个元素需要1000个bit位，1000/8=125字节。

## EXERCISE
题型: SINGLE_CHOICE
难度: MEDIUM
类别: INTERVIEW
题目: 以下哪种数据结构最适合实现一个需要频繁查找"是否存在"且不需要有序性的动态集合？
选项: ["A. 数组", "B. 链表", "C. 哈希表", "D. 平衡二叉搜索树"]
答案: C
解析: 哈希表提供平均O(1)的查找效率，且无需维护有序性，是最适合的选择。

## EXERCISE
题型: MULTIPLE_CHOICE
难度: MEDIUM
类别: INTERVIEW
题目: 关于哈希表实现集合，以下说法正确的有？
选项: ["A. 平均查找时间复杂度为 O(1)", "B. 最坏情况下查找退化为 O(n)", "C. 不支持范围查询", "D. 比平衡搜索树更节省空间"]
答案: A,B,C
解析: D不一定，哈希表的负载因子和冲突处理策略会影响空间利用率。在某些场景下平衡树可能更紧凑。

## EXERCISE
题型: FILL_BLANK
难度: MEDIUM
类别: POSTGRADUATE
题目: 位图实现两集合的并集操作使用位运算___完成，时间复杂度为 O(n/___)
答案: 或(OR),w
解析: 并集对应位的或运算，n为位数组长度，w为机器字长（通常32或64）。每个字（word）一次或运算处理w位。

## EXERCISE
题型: SINGLE_CHOICE
难度: HARD
类别: POSTGRADUATE
题目: C++ STL 中 set（基于红黑树）与 unordered_set（基于哈希表）的主要区别不包括？
选项: ["A. set 的元素有序，unordered_set 无序", "B. set 的插入/删除/查找为 O(log n)，unordered_set 为 O(1) 平均", "C. set 支持 lower_bound/upper_bound 范围查询", "D. set 占用的内存更少"]
答案: D
解析: D不是区别而是可能的情况，实际上set因维护红黑树的额外指针开销，内存占用通常比unordered_set更大。其他三项都是两者的真实区别。
