题目: 在一棵深度为 k 的满二叉树中，结点总数为？
选项: ["A. 2k", "B. 2^k", "C. 2^k - 1", "D. 2^(k+1) - 1"]

## EXERCISE
题型: SINGLE_CHOICE
难度: EASY
类别: BASIC
题目: 对一棵有 n 个结点的二叉树进行链式存储，空指针域的个数为？
选项: ["A. n", "B. n-1", "C. n+1", "D. 2n"]
答案: C
解析: 二叉链表共有 2n 个指针域。除根结点外每个结点被一个指针指向（n-1 个非空指针），其余指针为空，即 2n - (n-1) = n+1 个空指针域。

题目: 已知某二叉树的前序遍历序列为 ABDECF，中序遍历序列为 DBEACF，该二叉树的后序遍历序列是？
选项: ["A. DEBFCA", "B. DEBCFA", "C. DEBACF", "D. DEBAFC"]
答案: A
解析: 由前序确定根为 A，在中序中 A 的左边为左子树 DBE，右边为右子树 CF。对左子树：前序 BDE、中序 DBE，确定 B 为根，D 为 B 的左孩子，E 为 B 的右孩子。对右子树：前序 CF、中序 CF，确定 C 为根，F 为 C 的右孩子。后序为左-右-根，即 DEBFCA。

选项: ["A. 247", "B. 248", "C. 249", "D. 250"]
答案: B

## EXERCISE

题目: AVL树中结点的平衡因子定义为？
选项: ["A. 左子树高度 + 右子树高度", "B. 左子树高度 - 右子树高度", "C. 右子树高度 - 左子树高度", "D. 左子树结点数 - 右子树结点数"]
答案: B

答案: C
