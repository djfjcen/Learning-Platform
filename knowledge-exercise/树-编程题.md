## PROGRAMMING
题型: PROGRAMMING
难度: EASY
类别: BASIC
题目: 二叉树遍历重建
描述:
给定一棵二叉树的先序遍历和中序遍历序列（节点值用大写字母唯一标识），请重建该二叉树并输出其后序遍历序列。

输入说明：
一行，包含两个字符串 pre 和 in，以空格分隔，分别表示二叉树的先序遍历和中序遍历序列。

输出说明：
输出一行，为二叉树的后序遍历序列。

样例输入：
ABDECFG DBEAFCG

样例输出：
DEBFGCA

备注：
节点数量不超过 1000，每个节点的值是大写字母 A~Z 且不重复。

答案:
【C】
```c
#include <stdio.h>
#include <string.h>

char pre[1005], in[1005];

void post(int preL, int preR, int inL, int inR) {
    if (preL > preR) return;
    char root = pre[preL];
    int pos = inL;
    while (in[pos] != root) pos++;
    int leftLen = pos - inL;
    post(preL + 1, preL + leftLen, inL, pos - 1);
    post(preL + leftLen + 1, preR, pos + 1, inR);
    putchar(root);
}

int main() {
    scanf("%s %s", pre, in);
    int len = (int)strlen(pre);
    post(0, len - 1, 0, len - 1);
    printf("\n");
    return 0;
}
```


## PROGRAMMING

题目: 验证二叉搜索树
描述:
给定一棵二叉树，判断它是否是一棵合法的二叉搜索树（BST）。

二叉搜索树的定义：
1. 若左子树不空，则左子树上所有节点的值均小于其根节点的值。
2. 若右子树不空，则右子树上所有节点的值均大于其根节点的值。
3. 左右子树也分别为二叉搜索树。

输入说明：
第一行输入一个整数 n，表示树的节点数量。
接下来 n 行，每行包含三个整数 val left right，分别表示该节点的值、左孩子编号和右孩子编号。-1 表示空节点。根节点编号为 0。

输出说明：
若为二叉搜索树，输出 "true"；否则输出 "false"。

样例输入：
3
2 1 2
1 -1 -1
3 -1 -1

样例输出：
true

备注：
节点数量 n ≤ 1000，节点值在 int 范围内。

答案:
【C】
```c
#include <stdio.h>
#include <limits.h>

typedef struct { int val, left, right; } Node;
Node tree[1005];

int valid(int idx, long long min, long long max) {
    if (idx == -1) return 1;
    if (tree[idx].val <= min || tree[idx].val >= max) return 0;
    return valid(tree[idx].left, min, tree[idx].val) &&
           valid(tree[idx].right, tree[idx].val, max);
}

int main() {
    int n;
    scanf("%d", &n);
    for (int i = 0; i < n; i++)
        scanf("%d %d %d", &tree[i].val, &tree[i].left, &tree[i].right);

}
```

## PROGRAMMING
题型: PROGRAMMING
难度: MEDIUM
类别: BASIC
题目: 判断平衡二叉树
描述:
给定一棵二叉树，判断它是否是一棵平衡二叉树。

平衡二叉树的定义：二叉树中每个节点的左右子树高度差的绝对值不超过 1。

输入说明：
第一行输入一个整数 n，表示树的节点数量。
接下来 n 行，每行包含三个整数 val left right，分别表示该节点的值、左孩子编号和右孩子编号。-1 表示空节点。根节点编号为 0。

输出说明：
若为平衡二叉树，输出 "true"；否则输出 "false"。

样例输入：
5
1 1 2
2 3 4
3 -1 -1
4 -1 -1
5 -1 -1

样例输出：
true

备注：
节点数量 n ≤ 1000。

答案:
【C】
```c
#include <stdio.h>
#include <stdlib.h>

typedef struct { int val, left, right; } Node;
Node tree[1005];
int balanced;

int height(int idx) {
    if (idx == -1) return 0;
    int lh = height(tree[idx].left);
    int rh = height(tree[idx].right);
    if (abs(lh - rh) > 1) balanced = 0;
    return (lh > rh ? lh : rh) + 1;
}

int main() {
    int n;
    scanf("%d", &n);
    for (int i = 0; i < n; i++)
        scanf("%d %d %d", &tree[i].val, &tree[i].left, &tree[i].right);
    balanced = 1;
    height(0);
    printf(balanced ? "true\n" : "false\n");
    return 0;
}
```

## PROGRAMMING
题型: PROGRAMMING
难度: MEDIUM
类别: BASIC
题目: 哈夫曼树的带权路径长度
描述:
给定 n 个叶节点的权值，构造一棵哈夫曼树，求其带权路径长度（WPL）。

带权路径长度：从根节点到每个叶节点的路径长度与该叶节点权值的乘积之和。

输入说明：
第一行输入一个整数 n，表示叶节点的数量。
第二行输入 n 个整数，以空格分隔，表示每个叶节点的权值。

输出说明：
输出一个整数，表示最小带权路径长度 WPL。

样例输入：
5
1 2 3 4 5

样例输出：
33

备注：
1 ≤ n ≤ 1000，权值均为正整数且不超过 10000。

答案:
【C】
```c
#include <stdio.h>

int heap[2005];
int size;

void push(int x) {
    int i = size++;
    while (i > 0) {
        int p = (i - 1) / 2;
        if (heap[p] <= x) break;
        heap[i] = heap[p];
        i = p;
    }
    heap[i] = x;
}

int pop() {
    int ret = heap[0];
    int x = heap[--size];
    int i = 0;
    while (2 * i + 1 < size) {
        int a = 2 * i + 1, b = 2 * i + 2;
        if (b < size && heap[b] < heap[a]) a = b;
        if (heap[a] >= x) break;
        heap[i] = heap[a];
        i = a;
    }
    heap[i] = x;
    return ret;
}

int main() {
    int n;
    scanf("%d", &n);
    size = 0;
    for (int i = 0; i < n; i++) {
        int x;
        scanf("%d", &x);
        push(x);
    }
    int wpl = 0;
    while (size > 1) {
        int a = pop(), b = pop();
        int sum = a + b;
        wpl += sum;
        push(sum);
    }
    printf("%d\n", wpl);
    return 0;
}
```

答案:

```c
#include <stdio.h>

typedef struct { int val, left, right; } Node;
Node tree[1005];

int maxDepth(int idx) {
    if (idx == -1) return 0;
    int l = maxDepth(tree[idx].left);
    int r = maxDepth(tree[idx].right);
    return (l > r ? l : r) + 1;
}

int main() {

    scanf("%d", &n);
    for (int i = 0; i < n; i++)
        scanf("%d %d %d", &tree[i].val, &tree[i].left, &tree[i].right);
    printf("%d\n", n == 0 ? 0 : maxDepth(0));
    return 0;
}
```

## PROGRAMMING
题型: PROGRAMMING
难度: EASY
类别: BASIC
题目: 二叉搜索树的插入与查找
描述:
实现二叉搜索树的插入和查找操作，模拟一系列操作并输出每次查找的结果。

操作格式：
- insert x：向 BST 中插入值为 x 的节点。若 x 已存在则忽略。
- search x：判断值为 x 的节点是否在 BST 中。

输入说明：
第一行输入一个整数 n，表示操作次数。
接下来 n 行，每行为 "insert x" 或 "search x"。

输出说明：
对每个 search 操作，输出 "true" 或 "false"，每行一个。

样例输入：
5
insert 3
insert 1
search 1
search 2
insert 5

样例输出：
true
false

备注：
1 ≤ n ≤ 100000，x 为 int 范围内的整数。初始 BST 为空。

答案:
【C】
```c
#include <stdio.h>
#include <stdlib.h>
#include <string.h>

typedef struct Node {
    int val;
    struct Node *left, *right;
} Node;

Node* insert(Node* root, int x) {
    if (!root) {
        Node* n = (Node*)malloc(sizeof(Node));
        n->val = x; n->left = n->right = NULL;
        return n;
    }
    if (x < root->val) root->left = insert(root->left, x);
    else if (x > root->val) root->right = insert(root->right, x);
    return root;
}

int search(Node* root, int x) {
    if (!root) return 0;
    if (x == root->val) return 1;
    if (x < root->val) return search(root->left, x);
    return search(root->right, x);
}

int main() {
    int n;
    scanf("%d", &n);
    Node* root = NULL;
    char op[10];
    int x;
    while (n--) {
        scanf("%s %d", op, &x);
        if (strcmp(op, "insert") == 0)
            root = insert(root, x);
        else
            printf(search(root, x) ? "true\n" : "false\n");
    }
    return 0;
}
```

```python
    if root is None:

    if x < root.val:
        root.left = insert(root.left, x)
    elif x > root.val:
        root.right = insert(root.right, x)
    return root

def search(root, x):
    if root is None:
        return False
    if x == root.val:
        return True
    if x < root.val:
        return search(root.left, x)
    return search(root.right, x)

n = int(input())
root = None
for _ in range(n):
    parts = input().split()
    op, x = parts[0], int(parts[1])
    if op == 'insert':
        root = insert(root, x)
    else:
        print('true' if search(root, x) else 'false')
```
解析: 二叉搜索树的插入和查找都是 O(h) 的递归操作（h 为树高）。插入时若 key 小于当前节点则递归插入左子树，大于则递归插入右子树，相等则忽略。查找逻辑类似，找到则返回 true，到达空节点则返回 false。

输入说明：
第一行输入一个整数 n，表示数组长度。

=== 知识点ID: 18
=== 知识点名称: 哈夫曼树

## PROGRAMMING
题型: PROGRAMMING
难度: MEDIUM
类别: BASIC
题目: 前缀编码验证
描述:
哈夫曼编码是一种前缀编码——没有任何一个编码是另一个编码的前缀。给定一组二进制编码，判断这些编码是否构成合法的前缀编码。

输入说明：
第一行输入一个整数 n，表示编码数量。
接下来 n 行，每行由一个字符和一个二进制字符串组成，以空格分隔。字符为单个小写字母，二进制字符串仅包含 '0' 和 '1'。

输出说明：
若所有编码构成合法的前缀编码，输出 "true"；否则输出 "false"。

样例输入：
4
a 0
b 10
c 110
d 111

样例输出：
true

备注：
1 ≤ n ≤ 1000，每个二进制字符串长度不超过 1000。

答案:
【C】
```c
#include <stdio.h>
#include <string.h>

int child[10005][2];
int isEnd[10005];
int cnt;

int insert(char* s) {
    int cur = 0;
    for (int i = 0; s[i]; i++) {
        int c = s[i] - '0';
        if (isEnd[cur]) return 0;
        if (!child[cur][c]) child[cur][c] = ++cnt;
        cur = child[cur][c];
    }
    if (isEnd[cur] || child[cur][0] || child[cur][1]) return 0;
    isEnd[cur] = 1;
    return 1;
}

int main() {
    int n;
    scanf("%d", &n);
    memset(isEnd, 0, sizeof(isEnd));
    cnt = 0;
    int valid = 1;
    char ch, s[1005];
    for (int i = 0; i < n; i++) {
        scanf(" %c %s", &ch, s);
        if (valid && !insert(s)) valid = 0;
    }
    printf(valid ? "true\n" : "false\n");
    return 0;
}
```

【Python】
```python
class Trie:
    def __init__(self):
        self.child = {}
        self.is_end = False

    def insert(self, s):
        cur = self
        for ch in s:
            if cur.is_end:
                return False
            if ch not in cur.child:
                cur.child[ch] = Trie()
            cur = cur.child[ch]
        if cur.is_end or len(cur.child) > 0:
            return False
        cur.is_end = True
        return True

n = int(input())
root = Trie()
valid = True
for _ in range(n):
    parts = input().split()
    code = parts[1]
    if valid and not root.insert(code):
        valid = False
print('true' if valid else 'false')
```
解析: 使用字典树（Trie）验证前缀码：插入每个编码时检查两件事——当前路径上是否已有其他编码的终点（其他编码是当前编码的前缀），以及当前编码的终点是否已有子节点（当前编码是其他编码的前缀）。若两者均未违反，则该编码合法。
