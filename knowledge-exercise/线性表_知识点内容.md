=== 知识点ID: 11
=== 知识点名称: 顺序表

## OPERATION
标题: 顺序表插入操作
内容:
顺序表使用一段连续的存储单元依次存储数据元素，通常用数组实现。
在第 i 个位置（1 ≤ i ≤ n+1）插入新元素时：
第一步：判断表是否已满，若已满则无法插入。
第二步：检查插入位置是否合法，i 的有效范围为 1 到 n+1。
第三步：将第 i 个位置及之后的所有元素统一向后移动一个位置。
第四步：将新元素放入第 i 个位置，表长加一。

时间分析：
- 最好情况：在表尾插入（i = n+1），无需移动任何元素，O(1)。
- 最坏情况：在表头插入（i = 1），需要移动全部 n 个元素，O(n)。
- 平均情况：在等概率假设下，平均移动 n/2 个元素，O(n)。

标题: 顺序表删除操作
内容:
删除第 i 个位置的元素（1 ≤ i ≤ n）：
第一步：判断表是否为空，若为空则无法删除。
第二步：检查位置合法性。
第三步：将第 i+1 个位置及之后的所有元素向前移动一个位置，覆盖被删元素。
第四步：表长减一。

时间分析：
- 最好情况：删除表尾元素（i = n），无需移动，O(1)。
- 最坏情况：删除表头元素（i = 1），移动 n-1 个元素，O(n)。
- 平均情况：等概率下平均移动 (n-1)/2 个元素，O(n)。

标题: 顺序表按值查找
内容:
在顺序表中查找第一个值为 target 的元素的位置：
从表头开始逐个比较，找到则返回其位序；遍历完未找到则返回查找失败。
可设置哨兵：将目标值存入 0 号位置（或数组末尾多留一位），省去每次循环中的越界判断，
使循环条件简化为"未匹配到就继续"，减少约一半的比较次数。

## COMPLEXITY
标题: 顺序表各操作复杂度总结
内容:

| 操作 | 时间复杂度 | 说明 |
|------|-----------|------|
| 按位查找（随机访问） | O(1) | 数组下标直接定位 |
| 按值查找 | O(n) | 需要逐个比较 |
| 插入（指定位置） | O(n) 均值 | 需要移动元素 |
| 删除（指定位置） | O(n) 均值 | 需要移动元素 |
| 表尾追加 | O(1) 均摊 | 动态扩容时偶发 O(n) |

空间复杂度：O(n)，其中 n 为当前元素个数。
动态数组的容量管理：当元素个数达到容量上限时，通常按 2 倍策略扩容，
均摊每次插入的时间复杂度为 O(1)。

## CODE
标题: 动态顺序表完整实现
描述: 包含插入、删除、查找、扩容等核心操作的动态数组实现
语言: C++
代码:
#include <bits/stdc++.h>
using namespace std;

class SeqList {
private:
    int* data;
    int capacity;
    int size;

    void resize(int newCap) {
        int* newData = new int[newCap];
        for (int i = 0; i < size; i++)
            newData[i] = data[i];
        delete[] data;
        data = newData;
        capacity = newCap;
    }

public:
    SeqList(int initCap = 4) : size(0), capacity(initCap) {
        data = new int[capacity];
    }

    ~SeqList() { delete[] data; }

    bool insert(int pos, int val) {
        if (pos < 1 || pos > size + 1) return false;
        if (size >= capacity) resize(capacity * 2);
        for (int i = size; i >= pos; i--)
            data[i] = data[i - 1];
        data[pos - 1] = val;
        size++;
        return true;
    }

    bool remove(int pos, int& removedVal) {
        if (pos < 1 || pos > size || size == 0) return false;
        removedVal = data[pos - 1];
        for (int i = pos; i < size; i++)
            data[i - 1] = data[i];
        size--;
        return true;
    }

    int find(int target) const {
        for (int i = 0; i < size; i++)
            if (data[i] == target)
                return i + 1;
        return 0;
    }

    int get(int pos) const {
        if (pos < 1 || pos > size) throw out_of_range("Index out of range");
        return data[pos - 1];
    }

    int length() const { return size; }
    bool empty() const { return size == 0; }

    void print() const {
        cout << "[";
        for (int i = 0; i < size; i++) {
            cout << data[i];
            if (i < size - 1) cout << ", ";
        }
        cout << "]" << endl;
    }
};

int main() {
    SeqList list(4);
    list.insert(1, 10);
    list.insert(2, 20);
    list.insert(3, 30);
    list.print();

    list.insert(2, 15);
    cout << "插入15后: ";
    list.print();

    int removed;
    list.remove(3, removed);
    cout << "删除了 " << removed << ": ";
    list.print();

    cout << "查找20的位置: " << list.find(20) << endl;
    cout << "当前长度: " << list.length() << endl;
    return 0;
}

=== 知识点ID: 12
=== 知识点名称: 链表

## OPERATION
标题: 单链表基本概念与结构
内容:
单链表通过指针将一组离散的内存块串联起来，每个节点包含数据域和指针域。
节点结构：[数据 | next → 下一个节点的地址]
头节点：第一个数据节点之前的辅助节点，不存储实际数据，用于简化边界操作。
头指针：指向头节点（或首元节点）的指针，是链表的唯一入口。
尾节点：最后一个数据节点，其 next 指针为 NULL（或 nullptr）。
特点：不需要连续内存空间，插入和删除只需修改指针，但无法随机访问（必须从头遍历）。

标题: 单链表头插法
内容:
头插法将新节点插入到链表头部（头节点之后、原首元节点之前）。
步骤：
1. 创建新节点 s，填入数据。
2. 将 s->next 指向头节点的后继（即原来的首元节点）。
3. 将头节点的 next 指向 s。
特点：每插入一个元素都在头部，最终链表中元素的顺序与输入顺序**相反**。
时间复杂度：O(1)，仅需修改两个指针。

标题: 单链表尾插法
内容:
尾插法将新节点追加到链表尾部。
步骤：
1. 创建新节点 s，填入数据。
2. 若链表为空（仅头节点），将头节点的 next 指向 s。
3. 否则，将尾指针 r->next 指向 s，然后更新 r = s。
4. 最后令 s->next = NULL。
特点：保持元素的原始输入顺序。需要维护一个尾指针以避免每次都遍历到末尾（否则退化为 O(n)）。
时间复杂度：有尾指针时 O(1)，无尾指针需遍历至末尾则为 O(n)。

标题: 单链表按位删除
内容:
删除第 i 个位置的节点（1 ≤ i ≤ length）：
步骤：
1. 从头节点出发，工作指针 p 移动到第 i-1 个节点（待删节点的前驱）。
2. 若 p->next 为空（超出长度），删除失败。
3. 用临时指针 q 记录待删节点：q = p->next。
4. 将 p->next 绕过 q，指向 q->next（p->next = q->next）。
5. 释放 q 节点的内存：delete q。
关键：必须先找到前驱节点才能修改指针完成删除，因此需要从头遍历到第 i-1 个节点。
时间复杂度：O(n)（查找前驱需 O(n)，删除操作本身 O(1)）。

## COMPLEXITY
标题: 链表各操作复杂度总结
内容:

| 操作 | 单链表 | 双向链表 | 循环单链表 |
|------|--------|---------|-----------|
| 按位查找 | O(n) | O(n) | O(n) |
| 按值查找 | O(n) | O(n) | O(n) |
| 头部插入/删除 | O(1) | O(1) | O(1) |
| 尾部插入（有尾指针） | O(1) | O(1) | O(1) |
| 尾部删除 | O(n) | O(1)* | O(n) |
| 已知节点后插入 | O(1) | O(1) | O(1) |
| 已知节点删除 | O(n)** | O(1) | O(n)** |
| 空间开销 | 每节点额外1个指针 | 每节点额外2个指针 | 同单链表 |

*双向链表已知尾节点可直接操作前驱指针。
**单向链表删除已知节点仍需找前驱，除非用"复制后继法"变通处理。

空间复杂度：O(n)，每个节点额外占用一个指针域的空间（双向链表为两个）。

## CODE
标题: 完整单链表类实现
描述: 含头节点、尾插法、按位插入/删除、按值查找、遍历输出
语言: C++
代码:
#include <bits/stdc++.h>
using namespace std;

struct Node {
    int data;
    Node* next;
    Node(int val) : data(val), next(nullptr) {}
};

class LinkedList {
private:
    Node* head;
    int len;

public:
    LinkedList() : len(0) {
        head = new Node(0);   // 头节点
    }

    ~LinkedList() {
        Node* p = head;
        while (p) {
            Node* tmp = p->next;
            delete p;
            p = tmp;
        }
    }

    // 尾插法
    void pushBack(int val) {
        Node* newNode = new Node(val);
        Node* tail = head;
        while (tail->next) tail = tail->next;
        tail->next = newNode;
        len++;
    }

    // 头插法
    void pushFront(int val) {
        Node* newNode = new Node(val);
        newNode->next = head->next;
        head->next = newNode;
        len++;
    }

    // 按位插入（pos从1开始）
    bool insert(int pos, int val) {
        if (pos < 1 || pos > len + 1) return false;
        Node* p = head;
        for (int i = 0; i < pos - 1; i++)
            p = p->next;
        Node* newNode = new Node(val);
        newNode->next = p->next;
        p->next = newNode;
        len++;
        return true;
    }

    // 按位删除
    bool removeAt(int pos, int& removedVal) {
        if (pos < 1 || pos > len) return false;
        Node* p = head;
        for (int i = 0; i < pos - 1; i++)
            p = p->next;
        Node* q = p->next;
        removedVal = q->data;
        p->next = q->next;
        delete q;
        len--;
        return true;
    }

    // 按值查找，返回位序（从1开始），找不到返回0
    int find(int target) const {
        Node* p = head->next;
        int idx = 1;
        while (p) {
            if (p->data == target) return idx;
            p = p->next;
            idx++;
        }
        return 0;
    }

    int length() const { return len; }

    void print() const {
        Node* p = head->next;
        cout << "List(" << len << "): ";
        while (p) {
            cout << p->data;
            if (p->next) cout << " -> ";
            p = p->next;
        }
        cout << " -> NULL" << endl;
    }
};

int main() {
    LinkedList list;
    list.pushBack(10);
    list.pushBack(20);
    list.pushBack(30);
    list.print();

    list.pushFront(5);
    cout << "头插5后: ";
    list.print();

    list.insert(3, 15);
    cout << "在第3位插入15后: ";
    list.print();

    int removed;
    list.removeAt(2, removed);
    cout << "删除第2位的元素(" << removed << ")后: ";
    list.print();

    cout << "查找20的位置: " << list.find(20) << endl;
    return 0;
}

=== 知识点ID: 13
=== 知识点名称: 栈

## OPERATION
标题: 栈的基本概念
内容:
栈是一种限定仅在**表尾**进行插入和删除操作的线性表。
允许操作的一端称为**栈顶**（top），另一端称为**栈底**（bottom）。
不含任何元素的栈称为**空栈**。
栈的操作特性是 **LIFO**（Last In First Out，后进先出）：
最后压入栈的元素最先被弹出。
类比：一摞盘子，只能从最上面取放。

基本操作：
- Push（进栈/压栈）：在栈顶插入元素。
- Pop（出栈/弹栈）：删除并返回栈顶元素。
- Peek / Top（取栈顶）：查看栈顶元素但不删除。
- IsEmpty（判空）：判断栈是否为空。
- IsFull（判满）：针对顺序栈，判断是否已满。

标题: 顺序栈的实现
内容:
顺序栈用数组实现，用一个整型变量 top 记录栈顶位置。
两种常见约定：
方式一：top 初始为 -1，表示空栈。Push 时 top++ 再赋值，Pop 先取值再 top--。
方式二：top 初始为 0，表示空栈。Push 直接赋值再 top++，Pop 先 top-- 再取值。

栈满条件（方式一）：top == maxSize - 1
栈空条件（方式一）：top == -1
入栈：data[++top] = x
出栈：x = data[top--]

标题: 链式栈的实现
内容:
链式栈用单链表实现，栈顶设在链表头部（头节点之后的首元节点）。
优点：不存在栈满的问题（只要内存够），不需要预先分配固定大小。
入栈：新建节点 s，s->next = top->next，top->next = s —— O(1)
出栈：s = top->next，top->next = s->next，释放 s —— O(1)

标题: 栈的应用——表达式求值
内容:
利用栈可以方便地计算中缀表达式，经典算法是双栈法：
一个操作数栈（operand stack）和一个运算符栈（operator stack）。
扫描表达式：
- 遇到数字：压入操作数栈。
- 遇到运算符：与运算符栈顶元素比较优先级。
  - 当前运算符优先级更高：压入运算符栈。
  - 当前运算符优先级更低或相等：弹出栈顶运算符和两个操作数进行计算，结果压回操作数栈，重复比较。
- 遇到左括号'('：压入运算符栈。
- 遇到右括号')'：不断弹出运算符计算，直到遇到左括号为止。
最后运算符栈中剩余的运算符依次弹出计算，操作数栈中剩下的唯一结果即为答案。

## COMPLEXITY
标题: 栈各操作复杂度
内容:

| 操作 | 顺序栈 | 链式栈 |
|------|--------|--------|
| Push（入栈） | O(1) 均摊 | O(1) |
| Pop（出栈） | O(1) | O(1) |
| Peek（取栈顶） | O(1) | O(1) |
| 判空 | O(1) | O(1) |
| 判满 | O(1) | 不适用 |
| 空间复杂度 | O(n) 固定容量 | O(n) 按需分配 |

表达式求值：每个操作数和运算符最多进出栈一次，总时间复杂度 O(n)，n 为表达式中符号总数。

## CODE
标题: 顺序栈完整实现与应用
描述: 实现顺序栈，并用栈检测括号匹配
语言: C++
代码:
#include <bits/stdc++.h>
using namespace std;

template<typename T, int MAX_SIZE = 100>
class Stack {
private:
    T data[MAX_SIZE];
    int top;
public:
    Stack() : top(-1) {}

    bool isEmpty() const { return top == -1; }
    bool isFull() const { return top == MAX_SIZE - 1; }
    int size() const { return top + 1; }

    bool push(const T& x) {
        if (isFull()) return false;
        data[++top] = x;
        return true;
    }

    bool pop(T& x) {
        if (isEmpty()) return false;
        x = data[top--];
        return true;
    }

    bool peek(T& x) const {
        if (isEmpty()) return false;
        x = data[top];
        return true;
    }
};

// 应用：括号匹配检测
bool isBalanced(const string& expr) {
    Stack<char> st;
    unordered_map<char, char> match = {{')', '('}, {']', '['}, {'}', '{'}};
    set<char> open = {'(', '[', '{'};
    set<char> close = {')', ']', '}'};

    for (char c : expr) {
        if (open.count(c)) {
            st.push(c);
        } else if (close.count(c)) {
            char topChar;
            if (!st.peek(topChar) || topChar != match[c])
                return false;
            st.pop(topChar);
        }
    }
    return st.isEmpty();
}

// 应用：十进制转任意进制
string toBase(int num, int base) {
    if (num == 0) return "0";
    Stack<int> st;
    string digits = "0123456789ABCDEF";
    while (num > 0) {
        st.push(num % base);
        num /= base;
    }
    string result;
    char c;
    while (st.pop(c))
        result += digits[c];
    return result;
}

int main() {
    // 括号匹配测试
    cout << boolalpha;
    cout << "{[()]}" << " 匹配? " << isBalanced("{[()]}") << endl;
    cout << "{[(])}" << " 匹配? " << isBalanced("{[(])}") << endl;
    cout << "(()" << " 匹配? " << isBalanced("(()") << endl;

    // 进制转换测试
    cout << "255 转16进制 = " << toBase(255, 16) << endl;
    cout << "100 转8进制 = " << toBase(100, 8) << endl;
    cout << "42 转2进制 = " << toBase(42, 2) << endl;

    return 0;
}

=== 知识点ID: 14
=== 知识点名称: 队列

## OPERATION
标题: 队列的基本概念
内容:
队列是一种限定仅在**表尾插入**、**表头删除**的线性表。
允许插入的一端称为**队尾**（rear），允许删除的一端称为**队头**（front）。
队列的操作特性是 **FIFO**（First In First Out，先进先出）：
最先进入队列的元素最先离开。
类比：排队买票，先来的人先买到票离开。

基本操作：
- EnQueue（入队）：在队尾插入元素。
- DeQueue（出队）：删除并返回队头元素。
- GetFront（取队头）：查看队头元素但不删除。
- IsEmpty（判空）、IsFull（判满）。

标题: 顺序队列与假溢出问题
内容:
用数组实现的顺序队列存在"假溢出"问题：
即使数组前端有空闲位置，但 rear 已到达数组末尾，仍无法继续入队。
例如：数组大小为 5，经过几次出队后 front=3，rear=5（已满），
但实际上 0~2 这三个位置是空闲的——这就是假溢出。

解决方案：**循环队列**。
将数组看作首尾相接的环形结构，rear 到达末尾后回到开头继续使用。
取模运算：(rear + 1) % maxSize

标题: 循环队列的实现细节
内容:
循环队列的关键在于如何区分"空"和"满"：

方案一：牺牲一个存储单元
- 初始状态：front = 0, rear = 0
- 入队：data[rear] = x; rear = (rear + 1) % maxSize
- 出队：x = data[front]; front = (front + 1) % maxSize
- 队空条件：front == rear
- 队满条件：(rear + 1) % maxSize == front
- 有效元素个数：(rear - front + maxSize) % maxSize

方案二：增加 size 变量记录当前元素个数
- 队空条件：size == 0
- 队满条件：size == maxSize
- 无需牺牲存储单元

标题: 链式队列
内容:
链式队列用带头结点的单链表实现，队头在链表头部，队尾在链表尾部。
需要维护 front 和 rear 两个指针。
入队：新建节点接到 rear 之后，更新 rear 指针 —— O(1)
出队：删除 front 后的第一个节点，更新 front 指针 —— O(1)
优点：不存在假溢出和队满问题，空间按需分配。

## COMPLEXITY
标题: 队列各操作复杂度
内容:

| 操作 | 顺序队列（非循环） | 循环队列 | 链式队列 |
|------|-------------------|---------|---------|
| EnQueue（入队） | O(1) 但可能假溢出 | O(1) | O(1) |
| DeQueue（出队） | O(1) | O(1) | O(1) |
| GetFront（取队头） | O(1) | O(1) | O(1) |
| 判空 | O(1) | O(1) | O(1) |
| 空间利用率 | 低（假溢出浪费） | 高（循环利用） | 高（按需分配） |

BFS（广度优先搜索）中使用队列，每个节点入队一次出队一次，总时间复杂度 O(V+E)。

## CODE
标题: 循环队列完整实现
描述: 采用牺牲一个单元的方案，支持入队、出队、判空、判满、获取长度
语言: C++
代码:
#include <bits/stdc++.h>
using namespace std;

template<typename T, int MAX_SIZE = 10>
class CircularQueue {
private:
    T data[MAX_SIZE];
    int front, rear;

public:
    CircularQueue() : front(0), rear(0) {}

    bool isEmpty() const { return front == rear; }
    bool isFull() const { return (rear + 1) % MAX_SIZE == front; }
    int size() const { return (rear - front + MAX_SIZE) % MAX_SIZE; }

    bool enQueue(const T& x) {
        if (isFull()) return false;
        data[rear] = x;
        rear = (rear + 1) % MAX_SIZE;
        return true;
    }

    bool deQueue(T& x) {
        if (isEmpty()) return false;
        x = data[front];
        front = (front + 1) % MAX_SIZE;
        return true;
    }

    bool getFront(T& x) const {
        if (isEmpty()) return false;
        x = data[front];
        return true;
    }

    void print() const {
        cout << "Queue[" << size() << "/" << MAX_SIZE - 1 << "]: front=";
        int f = front;
        while (f != rear) {
            cout << data[f];
            f = (f + 1) % MAX_SIZE;
            if (f != rear) cout << " <- ";
        }
        cout << "= rear" << endl;
    }
};

int main() {
    CircularQueue<int> q;

    cout << "--- 入队 1~7 ---" << endl;
    for (int i = 1; i <= 7; i++) {
        q.enQueue(i);
        cout << "enQueue(" << i << ") -> ";
        q.print();
    }

    cout << "\n--- 出队 3 次 ---" << endl;
    int val;
    for (int i = 0; i < 3; i++) {
        q.deQueue(val);
        cout << "deQueue() = " << val << " -> ";
        q.print();
    }

    cout << "\n--- 再入队 8~9 ---" << endl;
    for (int i = 8; i <= 9; i++) {
        q.enQueue(i);
        cout << "enQueue(" << i << ") -> ";
        q.print();
    }

    cout << "\n队空? " << (q.isEmpty() ? "是" : "否") << endl;
    cout << "队满? " << (q.isFull() ? "是" : "否") << endl;
    cout << "当前长度: " << q.size() << endl;
    return 0;
}
