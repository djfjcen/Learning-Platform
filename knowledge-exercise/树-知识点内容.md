=== 知识点ID: 15
=== 知识点名称: 二叉树

## OPERATION
标题: 二叉树的链式存储结构
内容:
二叉树每个结点包含数据域、左孩子指针、右孩子指针。
n个结点的二叉链表中，共有2n个指针域，其中n-1个指向孩子结点，n+1个为空指针域。
利用这些空指针域可构造线索二叉树，加速遍历。

标题: 先序遍历（根左右）
内容:
访问根结点→先序遍历左子树→先序遍历右子树。
递归实现最直观；非递归实现借助栈，先将根入栈，每次出栈访问后依次将右孩子、左孩子入栈（确保左孩子先出栈）。

标题: 中序遍历（左根右）
内容:
中序遍历左子树→访问根结点→中序遍历右子树。
对二叉搜索树进行中序遍历，得到有序序列。
非递归实现：沿左链一路到底并入栈，出栈访问后转向右子树，重复此过程。

标题: 后序遍历（左根右）
内容:
后序遍历左子树→后序遍历右子树→访问根结点。
应用场景：计算目录大小（先算子目录再算父目录）、表达式树求值。
非递归实现较复杂，常用双栈法或标记法。

标题: 层序遍历
内容:
从上到下、从左到右逐层访问结点。
借助队列实现：根入队，每次出队访问后将左右孩子入队。
常用于求二叉树宽度、判断完全二叉树等场景。

## COMPLEXITY
标题: 时间复杂度与空间复杂度
内容:
四种遍历均访问每个结点恰好一次，时间复杂度均为O(n)。
递归遍历空间复杂度O(h)，h为树高，最坏情况单支树退化为O(n)。
层序遍历空间复杂度O(w)，w为树的最大宽度，最坏情况满二叉树最后一层约n/2个结点，O(n)。
链式存储空间复杂度O(n)，每个结点占用常量空间。

## CODE
标题: 二叉树链式存储与四种遍历
描述: 构建二叉树并进行先序、中序、后序（递归+非递归）和层序遍历
语言: C++
代码:
#include <bits/stdc++.h>
using namespace std;

struct TreeNode {
    int val;
    TreeNode *left, *right;
    TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
};

void preOrder(TreeNode* root) {
    if (!root) return;
    cout << root->val << " ";
    preOrder(root->left);
    preOrder(root->right);
}

void inOrder(TreeNode* root) {
    if (!root) return;
    inOrder(root->left);
    cout << root->val << " ";
    inOrder(root->right);
}

void postOrder(TreeNode* root) {
    if (!root) return;
    postOrder(root->left);
    postOrder(root->right);
    cout << root->val << " ";
}

void preOrderNR(TreeNode* root) {
    if (!root) return;
    stack<TreeNode*> s;
    s.push(root);
    while (!s.empty()) {
        TreeNode* p = s.top(); s.pop();
        cout << p->val << " ";
        if (p->right) s.push(p->right);
        if (p->left) s.push(p->left);
    }
}

void inOrderNR(TreeNode* root) {
    stack<TreeNode*> s;
    TreeNode* p = root;
    while (p || !s.empty()) {
        while (p) { s.push(p); p = p->left; }
        p = s.top(); s.pop();
        cout << p->val << " ";
        p = p->right;
    }
}

void levelOrder(TreeNode* root) {
    if (!root) return;
    queue<TreeNode*> q;
    q.push(root);
    while (!q.empty()) {
        TreeNode* p = q.front(); q.pop();
        cout << p->val << " ";
        if (p->left) q.push(p->left);
        if (p->right) q.push(p->right);
    }
}

int main() {
    TreeNode* root = new TreeNode(1);
    root->left = new TreeNode(2);
    root->right = new TreeNode(3);
    root->left->left = new TreeNode(4);
    root->left->right = new TreeNode(5);
    root->right->left = new TreeNode(6);

    cout << "先序: "; preOrder(root); cout << endl;
    cout << "中序: "; inOrder(root); cout << endl;
    cout << "后序: "; postOrder(root); cout << endl;
    cout << "先序(非递归): "; preOrderNR(root); cout << endl;
    cout << "中序(非递归): "; inOrderNR(root); cout << endl;
    cout << "层序: "; levelOrder(root); cout << endl;
    return 0;
}

=== 知识点ID: 16
=== 知识点名称: 二叉搜索树

## OPERATION
标题: 查找操作
内容:
从根结点开始，若目标值等于当前结点值，查找成功；若目标值小于当前结点值，递归查找左子树；若大于，递归查找右子树。
查找过程形成一条从根到目标结点的路径，比较次数不超过树的高度。

标题: 插入操作
内容:
先执行查找，若关键字已存在则插入失败（或根据需求更新）。
若不存在，查找失败的位置即为新结点的插入位置。
新结点一定作为叶子结点插入，不会破坏已有结构。

标题: 删除操作
内容:
分三种情况：(1)被删结点为叶子，直接删除；(2)被删结点只有左子树或只有右子树，用其孩子替代；(3)被删结点有左右子树，用其直接前驱（左子树最大结点）或直接后继（右子树最小结点）替换，再删除该前驱/后继。
第三种情况转化为前两种情况之一。

## COMPLEXITY
标题: 时间复杂度与空间复杂度
内容:
查找、插入、删除的平均时间复杂度为O(log n)，但最坏情况（插入有序序列形成单支树）退化为O(n)。
查找、插入的递归实现空间复杂度O(h)，最坏O(n)。
中序遍历可得到有序序列，时间复杂度O(n)。

## CODE
标题: BST插入与查找
描述: 构建二叉搜索树，实现递归/迭代查找和插入，中序遍历验证有序性
语言: C++
代码:
#include <bits/stdc++.h>
using namespace std;

struct BSTNode {
    int val;
    BSTNode *left, *right;
    BSTNode(int x) : val(x), left(nullptr), right(nullptr) {}
};

BSTNode* insert(BSTNode* root, int val) {
    if (!root) return new BSTNode(val);
    if (val < root->val)
        root->left = insert(root->left, val);
    else if (val > root->val)
        root->right = insert(root->right, val);
    return root;
}

BSTNode* search(BSTNode* root, int key) {
    if (!root || root->val == key) return root;
    return key < root->val ? search(root->left, key) : search(root->right, key);
}

BSTNode* searchNR(BSTNode* root, int key) {
    while (root && root->val != key)
        root = key < root->val ? root->left : root->right;
    return root;
}

BSTNode* findMin(BSTNode* root) {
    while (root && root->left) root = root->left;
    return root;
}

BSTNode* remove(BSTNode* root, int key) {
    if (!root) return nullptr;
    if (key < root->val)
        root->left = remove(root->left, key);
    else if (key > root->val)
        root->right = remove(root->right, key);
    else {
        if (!root->left) { BSTNode* t = root->right; delete root; return t; }
        if (!root->right) { BSTNode* t = root->left; delete root; return t; }
        BSTNode* succ = findMin(root->right);
        root->val = succ->val;
        root->right = remove(root->right, succ->val);
    }
    return root;
}

void inOrder(BSTNode* root) {
    if (!root) return;
    inOrder(root->left);
    cout << root->val << " ";
    inOrder(root->right);
}

int main() {
    BSTNode* root = nullptr;
    int arr[] = {5, 3, 7, 2, 4, 6, 8};
    for (int x : arr) root = insert(root, x);

    cout << "中序遍历: "; inOrder(root); cout << endl;
    cout << "查找4: " << (search(root, 4) ? "找到" : "未找到") << endl;
    cout << "查找9: " << (searchNR(root, 9) ? "找到" : "未找到") << endl;

    root = remove(root, 3);
    cout << "删除3后: "; inOrder(root); cout << endl;

    root = remove(root, 5);
    cout << "删除5后: "; inOrder(root); cout << endl;
    return 0;
}

=== 知识点ID: 17
=== 知识点名称: 平衡二叉树

## OPERATION
标题: 平衡因子与旋转
内容:
平衡因子 = 左子树高度 - 右子树高度。AVL树要求所有结点平衡因子绝对值不超过1。
插入或删除后，从被修改结点向上回溯，找到第一个失衡结点，对其进行旋转调整。

标题: LL旋转（右单旋）
内容:
失衡结点A的平衡因子为2，其左孩子B的平衡因子为1。
将B提升为子树根，A变为B的右孩子，B的原右子树变为A的左子树。
旋转后A和B的平衡因子均归0。

标题: RR旋转（左单旋）
内容:
失衡结点A的平衡因子为-2，其右孩子B的平衡因子为-1。
将B提升为子树根，A变为B的左孩子，B的原左子树变为A的右子树。
旋转后A和B的平衡因子均归0。

标题: LR旋转（先左后右双旋）
内容:
失衡结点A的平衡因子为2，其左孩子B的平衡因子为-1。
先在B处做左旋（变为LL型），再在A处做右旋。
新根为B的右孩子C，C的左子树变为B的右子树，C的右子树变为A的左子树。

标题: RL旋转（先右后左双旋）
内容:
失衡结点A的平衡因子为-2，其右孩子B的平衡因子为1。
先在B处做右旋（变为RR型），再在A处做左旋。
新根为B的左孩子C，C的左子树变为A的右子树，C的右子树变为B的左子树。

## COMPLEXITY
标题: 时间复杂度与空间复杂度
内容:
含有n个结点的AVL树，最大高度约为1.44×log₂(n+1)，最坏高度仍为O(log n)级别。
查找、插入、删除时间复杂度均为O(log n)，旋转操作只需修改少量指针，时间O(1)。
空间复杂度O(log n)（递归调用栈），每个结点额外存储高度或平衡因子，O(1)空间开销。

## CODE
标题: AVL树插入与旋转
描述: 含高度维护的AVL树插入实现，支持LL/RR/LR/RL四种旋转
语言: C++
代码:
#include <bits/stdc++.h>
using namespace std;

struct AVLNode {
    int val, height;
    AVLNode *left, *right;
    AVLNode(int x) : val(x), height(1), left(nullptr), right(nullptr) {}
};

int getHeight(AVLNode* n) { return n ? n->height : 0; }
void updateHeight(AVLNode* n) {
    n->height = 1 + max(getHeight(n->left), getHeight(n->right));
}

AVLNode* rotateRight(AVLNode* y) {
    AVLNode* x = y->left;
    AVLNode* T = x->right;
    x->right = y;
    y->left = T;
    updateHeight(y);
    updateHeight(x);
    return x;
}

AVLNode* rotateLeft(AVLNode* x) {
    AVLNode* y = x->right;
    AVLNode* T = y->left;
    y->left = x;
    x->right = T;
    updateHeight(x);
    updateHeight(y);
    return y;
}

int getBalance(AVLNode* n) {
    return n ? getHeight(n->left) - getHeight(n->right) : 0;
}

AVLNode* insert(AVLNode* node, int val) {
    if (!node) return new AVLNode(val);
    if (val < node->val)
        node->left = insert(node->left, val);
    else if (val > node->val)
        node->right = insert(node->right, val);
    else return node;

    updateHeight(node);
    int bal = getBalance(node);

    if (bal > 1 && val < node->left->val)   // LL
        return rotateRight(node);
    if (bal < -1 && val > node->right->val) // RR
        return rotateLeft(node);
    if (bal > 1 && val > node->left->val) { // LR
        node->left = rotateLeft(node->left);
        return rotateRight(node);
    }
    if (bal < -1 && val < node->right->val) { // RL
        node->right = rotateRight(node->right);
        return rotateLeft(node);
    }
    return node;
}

void inOrder(AVLNode* root) {
    if (!root) return;
    inOrder(root->left);
    cout << root->val << "(" << getBalance(root) << ") ";
    inOrder(root->right);
}

int main() {
    AVLNode* root = nullptr;
    int arr[] = {10, 20, 30, 40, 50, 25};
    for (int x : arr) root = insert(root, x);

    cout << "中序遍历(值(平衡因子)): "; inOrder(root); cout << endl;
    cout << "根结点: " << root->val << endl;
    cout << "树高: " << getHeight(root) << endl;
    return 0;
}

=== 知识点ID: 18
=== 知识点名称: 哈夫曼树

## OPERATION
标题: 哈夫曼树构建
内容:
给定n个带权叶子结点，构建最优二叉树（WPL最小）。
(1)将每个结点视为一棵单结点树，全部放入最小堆；(2)每次从堆中取出两棵根权值最小的树，合并为一棵新树（新根权值为两子树根权值之和）；(3)将新树放回堆中；(4)重复直到堆中只剩一棵树，即哈夫曼树。
构建过程共需n-1次合并。

标题: 哈夫曼编码
内容:
从根到叶子，左分支标记为0，右分支标记为1（或反过来），路径上的标记序列即为该叶子的哈夫曼编码。
哈夫曼编码是前缀编码——任何字符的编码都不是另一个字符编码的前缀，保证解码唯一确定。
权值越大的字符编码越短，实现最优压缩。

标题: 加权路径长度（WPL）
内容:
WPL = Σ(叶子权值 × 根到叶子的路径长度)。
哈夫曼树在所有带权二叉树中WPL最小。
n个叶子结点的哈夫曼树共有2n-1个结点，没有度为1的结点（正则二叉树）。

## COMPLEXITY
标题: 时间复杂度与空间复杂度
内容:
构建哈夫曼树使用最小堆，共进行n-1次合并，每次合并涉及两次删除最小值和一次插入，时间复杂度O(n log n)。
编码过程对每个叶子做一次路径遍历，总时间O(n)，编码总长度不超过O(n²)。
空间复杂度O(n)用于存储哈夫曼树（2n-1个结点）和编码表。

## CODE
标题: 哈夫曼树构建与编码
描述: 给定字符频次，构建哈夫曼树并生成前缀编码表
语言: C++
代码:
#include <bits/stdc++.h>
using namespace std;

unordered_map<char, string> huffmanCode;

struct HuffNode {
    char ch;
    int freq;
    HuffNode *left, *right;
    HuffNode(char c, int f) : ch(c), freq(f), left(nullptr), right(nullptr) {}
};

struct Cmp {
    bool operator()(HuffNode* a, HuffNode* b) { return a->freq > b->freq; }
};

void encode(HuffNode* root, string code) {
    if (!root) return;
    if (!root->left && !root->right)
        huffmanCode[root->ch] = code;
    encode(root->left, code + "0");
    encode(root->right, code + "1");
}

HuffNode* buildHuffman(unordered_map<char, int>& freq) {
    priority_queue<HuffNode*, vector<HuffNode*>, Cmp> pq;
    for (auto& p : freq)
        pq.push(new HuffNode(p.first, p.second));

    while (pq.size() > 1) {
        HuffNode* left = pq.top(); pq.pop();
        HuffNode* right = pq.top(); pq.pop();
        HuffNode* parent = new HuffNode('\0', left->freq + right->freq);
        parent->left = left;
        parent->right = right;
        pq.push(parent);
    }
    return pq.top();
}

int main() {
    unordered_map<char, int> freq = {
        {'a', 5}, {'b', 9}, {'c', 12}, {'d', 13}, {'e', 16}, {'f', 45}
    };

    HuffNode* root = buildHuffman(freq);
    encode(root, "");

    int wpl = 0;
    for (auto& p : huffmanCode) {
        cout << p.first << ": " << p.second << endl;
        wpl += freq[p.first] * p.second.length();
    }
    cout << "WPL = " << wpl << endl;
    return 0;
}
