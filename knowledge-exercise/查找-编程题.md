=== 知识点ID: 24
=== 知识点名称: 二分查找

## PROGRAMMING
题型: PROGRAMMING
难度: EASY
类别: BASIC
题目: 有序数组查找
描述:
给定一个升序排列的整数数组和一个目标值 key，请使用二分查找判断 key 是否存在。

输入说明：
第一行输入两个整数 n 和 key，表示数组长度和目标值。
第二行输入 n 个升序排列的整数。

输出说明：
若 key 存在，输出其下标；若不存在，输出 -1。下标从 0 开始。

样例输入：
6 9
1 3 5 7 9 11

样例输出：
4

备注：
1 <= n <= 100000，数组元素和 key 均为 int 范围内整数。

答案:
【C】
```c
#include <stdio.h>

int main() {
    int n, key;
    scanf("%d %d", &n, &key);
    int a[100000];
    for (int i = 0; i < n; i++) {
        scanf("%d", &a[i]);
    }

    int left = 0, right = n - 1;
    while (left <= right) {
        int mid = left + (right - left) / 2;
        if (a[mid] == key) {
            printf("%d", mid);
            return 0;
        } else if (a[mid] < key) {
            left = mid + 1;
        } else {
            right = mid - 1;
        }
    }

    printf("-1");
    return 0;
}
```

=== 知识点ID: 25
=== 知识点名称: 哈希表

## PROGRAMMING
题型: PROGRAMMING
难度: MEDIUM
类别: INTERVIEW
题目: 两数之和
描述:
给定 n 个整数和目标值 target，请判断是否存在两个不同位置的数，使它们的和等于 target。

输入说明：
第一行输入两个整数 n 和 target。
第二行输入 n 个整数。

输出说明：
若存在，输出 true；否则输出 false。

样例输入：
5 9
2 7 11 15 1

样例输出：
true

备注：
2 <= n <= 100000，数组元素和 target 均为 int 范围内整数。

答案:
【C】
```c
#include <stdio.h>
#include <stdlib.h>

#define SIZE 200003

typedef struct Node {
    int value;
    struct Node* next;
} Node;

int hashValue(int x) {
    long long v = x;
    if (v < 0) v = -v;
    return (int)(v % SIZE);
}

int contains(Node* table[], int x) {
    int idx = hashValue(x);
    for (Node* p = table[idx]; p != NULL; p = p->next) {
        if (p->value == x) return 1;
    }
    return 0;
}

void insert(Node* table[], int x) {
    int idx = hashValue(x);
    Node* node = (Node*)malloc(sizeof(Node));
    node->value = x;
    node->next = table[idx];
    table[idx] = node;
}

int main() {
    int n, target;
    scanf("%d %d", &n, &target);
    static Node* table[SIZE];

    for (int i = 0; i < n; i++) {
        int x;
        scanf("%d", &x);
        if (contains(table, target - x)) {
            printf("true");
            return 0;
        }
        insert(table, x);
    }

    printf("false");
    return 0;
}
```

【C++】
```cpp

```

【Python】
```python
n, target = map(int, input().split())

```

解析: 遍历数组时，用哈希表保存已经出现过的数。对当前数 x，只需 O(1) 平均时间判断 target - x 是否出现过。先查找再插入，可以保证两个数来自不同位置。
时间限制: 60
排序: 2

## PROGRAMMING
题型: PROGRAMMING
难度: MEDIUM
类别: EXAM
题目: 叶子链表范围查询
描述:
B+树的叶子结点按关键字升序连接。现在给出若干个已经升序排列的叶子结点，请输出区间 [L, R] 内的所有关键字。

输入说明：
第一行输入两个整数 m 和 k，表示叶子结点个数和每个叶子结点中的关键字个数。
接下来 m 行，每行输入 k 个升序整数，且整体从前到后也是升序的。
最后一行输入两个整数 L 和 R。

输出说明：
按升序输出所有位于 [L, R] 的关键字，关键字之间用一个空格分隔。
若没有符合条件的关键字，输出 EMPTY。

样例输入：
3 3
3 8 12
18 21 29
35 40 48
10 40

样例输出：
12 18 21 29 35 40

备注：
1 <= m, k <= 1000，所有关键字均为 int 范围内整数。

答案:
【C】
```c
#include <stdio.h>

int main() {
    int m, k;
    scanf("%d %d", &m, &k);
    int total = m * k;
    int printed = 0;
    int values[1000000];

    for (int i = 0; i < total; i++) {
        scanf("%d", &values[i]);
    }

    int left, right;
    scanf("%d %d", &left, &right);

    for (int i = 0; i < total; i++) {
        if (values[i] >= left && values[i] <= right) {
            if (printed) printf(" ");
            printf("%d", values[i]);
            printed = 1;
        }
        if (values[i] > right) break;
    }

    if (!printed) printf("EMPTY");
    return 0;
}
```

【C++】
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int m, k;
    cin >> m >> k;
    vector<int> values(m * k);
    for (int& x : values) cin >> x;

    int left, right;
    cin >> left >> right;

    bool printed = false;
    for (int x : values) {
        if (x >= left && x <= right) {
            if (printed) cout << ' ';
            cout << x;
            printed = true;
        }
        if (x > right) break;
    }

    if (!printed) cout << "EMPTY";
    return 0;
}
```

【Python】
```python
m, k = map(int, input().split())
values = []
for _ in range(m):
    values.extend(map(int, input().split()))

left, right = map(int, input().split())
ans = []
for x in values:
    if left <= x <= right:
        ans.append(str(x))
    if x > right:
        break

print(' '.join(ans) if ans else 'EMPTY')
```

解析: B+树的叶子结点按关键字顺序连接，范围查询可以从第一个可能位置开始沿叶子链表顺序扫描，直到关键字超过右边界。本题用已经展开的叶子层模拟这一过程。

排序: 3
