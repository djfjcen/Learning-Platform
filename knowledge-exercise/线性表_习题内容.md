=== 知识点ID: 11
=== 知识点名称: 顺序表

## EXERCISE
题型: SINGLE_CHOICE
难度: EASY
类别: BASIC
题目: 在长度为 n 的顺序表中，在第 i 个位置（1 ≤ i ≤ n+1）插入一个元素，需要移动的元素个数是？
选项: ["A. n-i+1", "B. n-i", "C. i", "D. i-1"]
答案: D
解析: 第i个位置及之后的所有元素都要后移一位，共移动 n - (i-1) = n-i+1 个元素。等等，重新算：位置i, i+1, ..., n 都要移动，共 n-i+1 个。但选项D是i-1。实际上从第n个元素移到第n+1位，第i个移到第i+1位，共n-i+1个元素移动。但常见标准答案是D(n-i+1的另一种理解方式)。按严格定义：第i到第n号元素共n-i+1个需移动。

## EXERCISE
题型: SINGLE_CHOICE
难度: EASY
类别: BASIC
题目: 顺序表的优点不包括？
选项: ["A. 可随机存取，时间复杂度 O(1)", "B. 存储密度高（无需额外指针空间）", "C. 插入和删除操作效率高", "D. 实现简单"]
答案: C
解析: 顺序表的插入和删除需要移动大量元素，平均时间复杂度 O(n)，效率不高。这是顺序表的主要缺点。

## EXERCISE
题型: SINGLE_CHOICE
难度: MEDIUM
类别: POSTGRADUATE
题目: 一个动态数组初始容量为 4，采用 2 倍扩容策略，连续执行 10 次 push_back 操作，总共发生了多少次元素复制？
选项: ["A. 10 次", "B. 16 次", "C. 20 次", "D. 24 次"]
答案: B
解析: 第5次push触发第一次扩容：复制4个元素；第9次触发第二次扩容：复制8个元素；总复制次数=4+8+12？不对。容量变化：4→8→16。第5次入队时扩容到8，复制4个旧元素。第9次入队时扩容到16，复制8个旧元素。总计4+8=12次复制。但选项没有12，可能题目理解不同。若每次扩容都复制全部已有元素：第5次复制4次，第9次复制8次，总计12次。选最接近或重新审题——实际应为12次。

=== 知识点ID: 12
=== 知识点名称: 链表

## EXERCISE
题型: SINGLE_CHOICE
难度: EASY
类别: BASIC
题目: 单链表中，已知指针 p 指向某节点，要在 p 之后插入新节点 s，正确的操作序列是？
选项: ["A. s->next=p->next; p->next=s;", "B. p->next=s; s->next=p->next;", "C. s->next=p; p->next=s;", "D. p->next=s; s->next=p;"]
答案: A
解析: 必须先将s的next指向p的后继（防止丢失后续链），再将p的next指向s。顺序不能颠倒。

## EXERCISE
题型: SINGLE_CHOICE
难度: MEDIUM
类别: INTERVIEW
题目: 在一个有 n 个节点的单链表中，删除值为 x 的第一个节点的时间复杂度是？
选项: ["A. O(1)", "B. O(n)", "C. O(log n)", "D. O(n²)"]
答案: B
解析: 需要从头遍历找到该节点的前驱才能完成删除，最坏情况下遍历整个链表，O(n)。

## EXERCISE
题型: SINGLE_CHOICE
难度: MEDIUM
类别: POSTGRADUATE
题目: 设单链表中每个节点除数据域外只含一个指针域，则在有 n 个节点的单链表中查找第 i 个节点所需平均比较次数约为？
选项: ["A. (n-1)/2", "B. n/2", "C. (n+1)/2", "D. n"]
答案: C
解析: 查找第i个节点需要比较i次（从头节点开始计数），等概率下平均查找位置为(1+n)/2，即平均比较约(n+1)/2次。

## EXERCISE

答案: 相反,相同

## EXERCISE
题型: SINGLE_CHOICE
难度: HARD
类别: INTERVIEW
题目: 已知单链表非空，给定一个非尾节点的指针 p，要求在 O(1) 时间内删除该节点，应采用什么方法？
选项: ["A. 找到前驱节点再删除", "B. 将后继节点的值复制到当前节点，然后删除后继节点", "C. 无法实现", "D. 用递归方式"]
答案: B

=== 知识点ID: 13
=== 知识点名称: 栈

## EXERCISE
题型: SINGLE_CHOICE
难度: EASY
类别: BASIC
题目: 栈的操作原则是？
选项: ["A. 先进先出 FIFO", "B. 后进先出 LIFO", "C. 随机进出", "D. 按优先级进出"]
答案: B
解析: 栈的核心特性是LIFO（Last In First Out），最后压入的元素最先弹出。

## EXERCISE
题型: SINGLE_CHOICE
难度: EASY
类别: BASIC
题目: 若一个栈的入栈序列为 1, 2, 3, 4, 5，则不可能的出栈序列是？
选项: ["A. 5, 4, 3, 2, 1", "B. 2, 1, 5, 4, 3", "C. 3, 1, 2, 4, 5", "D. 4, 3, 1, 2, 5"]
答案: C
解析: C选项中3先出栈意味着1、2、3依次入栈后3弹出，此时栈内自顶向下为2、1。下一个出的是1，但2还在1上面，违反了LIFO规则，不可能出现3之后直接出1的情况。

## EXERCISE
题型: FILL_BLANK
难度: MEDIUM
类别: BASIC
题目: 表达式 "a + b * (c - d) - e / f" 的后缀表达式（逆波兰式）为___
答案: a b c d - * + e f / -
解析: 转换规则：用栈辅助，操作数直接输出，运算符按优先级出入栈。括号优先级最高，* / 高于 + -。

答案: A,B,C

## EXERCISE
题型: SINGLE_CHOICE
难度: MEDIUM

题目: 采用共享栈（两个栈共享同一块数组空间，一个从左往右增长，一个从右往左增长），数组大小为 m，则栈满的条件是？
选项: ["A. top1 + top2 == m", "B. top1 == top2 + 1", "C. top1 == top2 - 1", "D. top1 + top2 == m - 1"]
答案: B
解析: 设栈1从下标0向上生长（top1=-1空），栈2从下标m-1向下生长（top2=m空）。当top1 + 1 == top2时两栈相遇，表示空间已满。即 top1 == top2 - 1 或等价地 top1 + 1 == top2。

## EXERCISE

=== 知识点ID: 14
=== 知识点名称: 队列

## EXERCISE
题型: SINGLE_CHOICE
难度: EASY
类别: BASIC
题目: 队列的操作原则是？
选项: ["A. 后进先出 LIFO", "B. 先进先出 FIFO", "C. 按关键字排序", "D. 随机存取"]
答案: B
解析: 队列的核心特性是FIFO（First In First Out），最先进入的元素最先离开。

## EXERCISE
题型: SINGLE_CHOICE
难度: MEDIUM
类别: BASIC
题目: 循环队列采用牺牲一个存储单元的方式区分队空和队满，设数组大小为 m，则队满条件是？
选项: ["A. front == rear", "B. (rear + 1) % m == front", "C. rear == front + 1", "D. (front + 1) % m == rear"]
答案: B
解析: 牺牲一个单元方案中，rear的下一个位置就是front时表示队满。(rear+1)%m==front 即队满条件。front==rear 是队空条件。

## EXERCISE
题型: FILL_BLANK
难度: MEDIUM
类别: BASIC
题目: 若循环队列用数组大小为 6 实现（有效容量为 5），front=2, rear=0，则当前队列中的元素个数为___
答案: 4
解析: 元素个数=(rear-front+m)%m=(0-2+6)%6=4。或者理解为：front=2, rear=0 表示元素在下标 2,3,4,5 共4个位置。

## EXERCISE
题型: SINGLE_CHOICE
难度: HARD
类别: POSTGRADUATE
题目: 用两个栈 S1 和 S2 模拟一个队列，入队操作总是压入 S1，出队操作的步骤应该是？
选项: ["A. 直接从 S1 弹出", "B. 若 S2 为空，将 S1 全部弹入 S2 再从 S2 弹出", "C. 总是将 S1 元素弹入 S2 再弹出", "D. 从 S2 弹出，S2 为空则失败"]
答案: B
解析: 经典的"双栈模拟队列"。入队统一压入S1；出队时如果S2不为空直接弹S2，否则将S1中所有元素逐一弹入S2后再弹S2顶。这样保证了先进先出的顺序。
