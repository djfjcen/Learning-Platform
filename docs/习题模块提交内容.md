# 习题内容提交规范（选择 / 填空专用）
> 适用对象：`exercise` 表  
> 生效范围：**仅选择题、填空题**（不含编程题）  


---

## 一、适用范围

✅ 支持的题型（仅以下三种）：
- `SINGLE_CHOICE`：单选题
- `MULTIPLE_CHOICE`：多选题
- `FILL_BLANK`：填空题

❌ 不包含：
- `PROGRAMMING`（编程题）
- 任何统计类字段（通过率、尝试次数等）

---

## 二、禁止提交的内容（硬性）

| 禁止项 | 原因 |
|------|------|
| 一级模块节点习题（ID 1~6） | 这些为分类节点，不直接挂题 |
| 缺失题干 / 答案 / 解析 | 表结构中均为 NOT NULL |
| 多选题答案不带逗号 | 需明确分隔，便于解析 |
| 空题目、无意义占位 | 不符合内容审核要求 |

---

## 三、表字段 → 提交字段映射（严格一致）

| exercise 表字段 | 是否必填 | 提交要求 |
|----------------|---------|---------|
| `knowledge_point_id` | ✅ | 数字，必须使用知识点 ID |
| `title` | ✅ | 题目正文 |
| `type` | ✅ | `SINGLE_CHOICE` / `MULTIPLE_CHOICE` / `FILL_BLANK` |
| `difficulty` | ✅ | `EASY` / `MEDIUM` / `HARD` |
| `category` | ✅ | `BASIC` / `POSTGRADUATE` / `INTERVIEW` |
| `options` | ⚠️ 选择题必填 | JSON 数组字符串 |
| `answer` | ✅ | 文本，详见下方规则 |
| `analysis` | ✅ | 文本 |
| `time_limit` | ❌ | 按难度自动生成 |
| `description` | ❌ | 选择 / 填空不使用 |
| `initial_code` | ❌ | 不使用 |
| `test_cases` | ❌ | 不使用 |
| `pass_rate` | ❌ | 不提交 |
| `total_attempts` | ❌ | 不提交 |
| `correct_attempts` | ❌ | 不提交 |

---

## 四、答案格式规则（**仅此规范有效**）

### 1️⃣ 单选题
```
答案: A
```

### 2️⃣ 多选题
```
答案: A,B
答案: A,B,D
```
✅ **必须英文逗号，无空格**

### 3️⃣ 填空题
```
答案: 输入,输出,确定性
答案: 存储,运算,传输,加密
```
✅ **数量不限**
✅ **顺序与题目中 `___` 一一对应**


---

## 五、提交格式（标准模板）

```
=== 知识点ID: 12
=== 知识点名称: 链表

## EXERCISE
题型: SINGLE_CHOICE
难度: EASY
类别: BASIC
题目: 单链表中，访问第 i 个元素的时间复杂度是？
选项: ["A. O(1)", "B. O(n)", "C. O(log n)", "D. O(n²)"]
答案: B
解析: 单链表不支持随机访问，需要从头遍历到第 i 个节点。


## EXERCISE
题型: MULTIPLE_CHOICE
难度: MEDIUM
类别: INTERVIEW
题目: 下列哪些结构是线性结构？
选项: ["A. 栈", "B. 队列", "C. 二叉树", "D. 图"]
答案: A,B
解析: 栈和队列是受限线性表，二叉树和图为非线性结构。


## EXERCISE
题型: FILL_BLANK
难度: EASY
类别: BASIC
题目: 算法的五个重要特性是___、___、___、___和___。
答案: 输入,输出,确定性,可行性,有穷性
解析: 算法必须满足这五个基本特性，否则不能称为算法。

```

## 七、常见错误（直接驳回）

❌ 多选题答案写成 `ABD`  
✅ 正确：`A,B`


❌ 知识点ID 写名称  
✅ 正确：数字 ID
